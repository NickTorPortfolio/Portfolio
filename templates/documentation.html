Glossary of Terms

Definitions of the various mathematical terms that can be encountered in the program output are provided below.

Even and Odd Numbers

An even number is an integer number that can be represented as product of some integer and 2. For example, 6 is an even number because 6 = 3 × 2.

An odd number is an integer number that can not be represented as product of some integer and 2, but can be represented as 1 plus product of some integer and 2. For example, 7 is an odd number because 7 = 1 + 3 × 2.

Divisors. Prime and Composite Numbers

A divisor of an integer N is such iteger M, that for some other integer K (which is also a divisor), the following holds true: N=MK. For example, divisors of 6 are 1, 2, 3 and 6: 6=1×6=2×3. For any integer N, number N itself and 1 are always divisors of N.

A prime number is a number greater than 1 whose only divisors are number itself and 1. Aside from 1, all other integers are called composite. Number 1 is neither prime nor composite.

Two numbers are called coprime in relation to each other if they don't share any divisors in common aside from 1.

Divisor K is an unitary proper divisor of N if K and N/K are coprimes and K isn't equal to N.

Representation of an integer n purely in terms of product of powers of prime numbers is called prime factorization of n. Prime factorization of each number is unique. For example, prime factorization of 60 is 22×3×5.

If number has no powers greater than 1 in its prime factorization representation, it is called a squareless number.

Number is called a powerful number if and only if it can be represented as a positive integer power of an integer number.

Number is called a semiprime number if and only if can be reprsented as product of two, not neccessary distinct, prime numbers.

Number is called a sphenic number if and only if it can be reprsented as product of three distinct prime numbers.

Number is called a pronic number if and only if it can be reprsented as product of two consequent integers.

The Number of Divisors and Sum of Divisors

The sum of all divisors of an integer n is denoted σ(n). The ratio σ(n)/n is known as an abundancy index.

If σ(n)=2n, then number n is called a perfect number. If σ(n) > 2n, then number n is abundant, likewise if σ(n) < 2n, then number n is deficient.

Number is called a refactorable number if and only if it is disible by the number of its divisors.

Number is called an arithmetic number if and only if its sum of divisor is disible by the number of its divisors.
